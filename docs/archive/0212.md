# 1

flutter_soloud 覆盖了所有主流平台，且由于它基于 C++ (SoLoud) 和 Dart FFI，在不同平台上的表现非常一致：平台是否支持备注macOS💙 支持兼容 10.15+，在你的 macOS 12.7 上跑得非常稳。Android💙 支持兼容 API 21+，支持 AAudio (Android 11+) 自动切换。iOS💙 支持兼容 iOS 13.0+。Windows💙 支持基于 Miniaudio 后端，性能极佳。Linux💙 支持支持主流发行版（需安装 ALSA 库）。Web💙 支持通过 WASM 实现，支持基本播放功能。
